                      TROUBLESHOOTING
================================================================================

Q: "Cannot find module 'express'"
A: Run: npm install

Q: "ECONNREFUSED" MongoDB error
A: Start MongoDB service first:
   Windows: services.msc â†’ Start MongoDB
   Mac: brew services start mongodb-community
   Linux: sudo systemctl start mongod

Q: "Port 5000 already in use"
A: Change PORT in .env to 5001 or kill process using port 5000

Q: "CORS error in browser"
A: Clear browser cache and hard refresh (Ctrl+F5)

A: Clear browser localStorage and login again
